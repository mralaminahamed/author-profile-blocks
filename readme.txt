Showcase author profiles with a beautiful Gutenberg block that displays author information from a custom post type.

== Description ==

Author Profile Showcase is a modern WordPress plugin that allows you to create and display author profiles using a custom Gutenberg block. It's perfect for showcasing team members, contributors, staff, or any individuals you want to feature on your website.

The plugin creates a dedicated custom post type for author profiles, making it easy to manage all your author information in one place. Then, using the Author Profile block, you can display these profiles anywhere on your website with full customization options.

**Key Features:**

* **Custom Post Type for Author Profiles**: Store and manage all author information in one dedicated location
* **User-Friendly Gutenberg Block**: Easily add author profiles to any page or post
* **Flexible Display Options**: Control image display, email visibility, and description text
* **Customizable Styling**: Change background colors, text alignment, padding, and more
* **Additional Content Area**: Add supplementary information about the author with the "Show More" section
* **Fully Responsive Design**: Looks great on all devices and screen sizes
* **Translation-Ready**: Fully internationalized with .pot file included
* **Accessibility-Focused**: Built with web accessibility best practices

The Author Profile custom post type includes fields for:

* Author Name
* Email Address
* Profile Image (using WordPress featured image)
* Detailed Description

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/author-profile-showcase` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Create author profiles by navigating to Author Profiles > Add New in your WordPress admin.
4. Use the Author Profile block in any post or page to display your authors.

== Frequently Asked Questions ==

= How do I add a new author profile? =

Navigate to Author Profiles > Add New in your WordPress admin. Fill in the author's name, email, description, and add a featured image. All profiles are managed in this central location.

= Can I customize how the author profiles appear? =

Yes! The Author Profile block includes many customization options in the block sidebar:
* Toggle image, email, and description visibility
* Change background color
* Adjust text alignment (left, center, right)
* Modify padding
* Add additional content in a "Show More" section

= Is the block responsive? =

Yes, the Author Profile block is designed to look great on all screen sizes from mobile devices to desktop computers. The layout automatically adjusts based on the available screen space.

= Can I display multiple authors on a page? =

Yes, you can add as many Author Profile blocks as you need on any page or post. Each block can display a different author profile with its own customization settings.

= Does this plugin work with my theme? =

The plugin is designed to work with any WordPress theme that supports Gutenberg blocks (which includes most modern themes). The styling is contained within the block to ensure compatibility.

= How can I translate the plugin? =

The plugin is fully internationalized and includes a .pot file in the languages directory. You can use any WordPress translation plugin to translate the text strings.
