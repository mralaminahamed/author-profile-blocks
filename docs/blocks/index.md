# Author Profile Blocks
{: .no_toc }

Detailed documentation for each of the Author Profile Blocks.
{: .fs-6 .fw-300 }

## Available Blocks

The plugin includes four main blocks for showcasing author information:

1. [Author Profile Block]({{ site.baseurl }}{% link blocks/author-profile.md %}) - Display a single author with detailed information
2. [Author Grid Block]({{ site.baseurl }}{% link blocks/author-grid.md %}) - Present multiple authors in a responsive grid layout
3. [Author Carousel Block]({{ site.baseurl }}{% link blocks/author-carousel.md %}) - Showcase multiple authors in an interactive carousel
4. [Author List Block]({{ site.baseurl }}{% link blocks/author-list.md %}) - Display multiple authors in a customizable list format

## Common Features

All blocks share several common features and settings:

### Display Options

- **Author Image** - Show/hide the author's profile picture
- **Author Position** - Show/hide the author's position/title
- **Author Email** - Show/hide the author's email address
- **Author Description** - Show/hide the author's biographical information
- **Member Since Date** - Show/hide when the author joined, with customizable label
- **Social Links** - Show/hide the author's social media profiles

### Style Options

- **Background Color** - Set a custom background color for the block
- **Text Alignment** - Align text left, center, or right
- **Padding** - Adjust the padding around the content
- **Shadow** - Add a subtle shadow effect (Grid, Profile, and Carousel blocks)
- **Border** - Add a customizable border (Grid, Profile, and Carousel blocks)
- **Rounded Corners** - Add rounded corners to the block
- **Hover Effects** - Add interactive hover animations (List and Carousel blocks)

### Layout Options

All blocks offer multiple layout options:

- **Card** - Information displayed in a card format
- **Compact** - A more condensed layout
- **Centered** - Content centered for a more formal presentation (Grid, Profile, and Carousel blocks)
- **Detailed** - Expanded layout with more information (List block)

## Usage Tips

- Use the **Author Profile Block** for detailed author bios on single author pages
- Use the **Author Grid Block** for team or contributor pages with a visual gallery layout
- Use the **Author Carousel Block** for featured authors on home pages or sidebars
- Use the **Author List Block** for team directories or contributor listings that benefit from a vertical format
