## Image Conventions

Please follow these conventions when creating images:

1. **Resolution**: Use a minimum resolution of 1280×800 pixels for screenshots
2. **Format**: Use PNG format for screenshots and UI elements
3. **Compression**: Optimize all images for web
4. **Consistency**: Maintain consistent size and style across similar images
5. **Annotations**: Add annotations when necessary to highlight important elements
6. **Theme**: Use the default WordPress admin theme for admin screenshots
7. **Content**: Use appropriate placeholder content that doesn't include personal information

## Creating Images

To create high-quality documentation images:

1. Set up a clean WordPress installation
2. Use the Twenty Twenty-Four theme (or current default theme)
3. Create sample content with appropriate lorem ipsum text
4. Configure blocks with visually appealing settings
5. Take screenshots at consistent sizes
6. Crop and optimize images before adding to this directory

## Image Updates

When updating documentation images:

1. Maintain consistent naming for easier updates
2. Update images when UI changes occur
3. Create new images when new features are added
4. Remove deprecated images that no longer represent current functionality
