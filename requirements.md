# Author Profile Blocks - Project Requirements

## 🧪 Core Requirements

Create a Gutenberg block that displays author information from a custom post type called `author_profile`.

### 1. Register `author_profile` Custom Post Type

**Fields:**
- Name (title)
- Email (custom field)
- Image (featured image or custom field)
- Description (custom field)

### 2. Author Profile Block

- Add a block called "Author Profile"
- Allow the editor to search and select an author from existing `author_profile` posts
- Display the selected author's:
  - Image
  - Name
  - Email
  - Description

### 3. Inspector Controls

- Toggle: "Show More Section"
  - If enabled, show an editable RichText paragraph below the card
- Background color
- Text alignment
- Padding

## 🎯 Notes
